## 摘要
近年来，基于扩散的模型使得在实时流式传输中生成逼真的音频驱动虚拟形象成为可能。然而，现有方法难以维持视觉时间一致性，并且在复杂的交互式流式场景中无法显式感知用户意图。为了解决这些挑战，我们提出了InteractiveAvatar，一个实时无限流式视频生成框架，支持视觉一致的虚拟形象视频生成和意图感知的交互。通过自回归蒸馏，InteractiveAvatar能够在任意长时间内实现人类虚拟形象的实时流式生成。对于视觉一致性，我们引入了长-短视觉记忆（LSVM）机制，该机制将历史视觉信息灵活压缩为紧凑的标记，同时保留了短程连贯性和长期一致性。为了生成与用户意图一致的带有语音和动作的虚拟形象，我们提出了推理-反应模块（RRM），其中包含状态循环策略和缓存切换机制。在多种场景下的广泛实验结果表明，我们的方法在长时间生成中实现了最先进的视觉一致性，同时支持实时的复杂用户-虚拟形象交互。

## Abstract
Recent diffusion-based models have enabled realistic audio-driven avatar generation in real-time streaming. However, existing approaches struggle to maintain visual temporal consistency and fail to explicitly perceive user intent in complex interactive streaming scenarios. To address these challenges, we propose InteractiveAvatar, a real-time infinite-streaming video generation framework that supports visually consistent avatar video generation and intent-aware interactions. With autoregressive distillation, InteractiveAvatar achieves real-time str-eaming generation of human avatars over arbitrarily long durations. For visual consistency, we introduce a Long-Short Visual Memory (LSVM) mechanism that flexibly compresses historical visual information into compact tokens, preserving both short-range coherence and long-term consistency. To generate avatars with speeches and actions aligned with user intent, we propose a Reasoning-Reaction Module (RRM), which incorporates a State-Cycling strategy and a Cache-Switching mechanism. Extensive experimental results over diverse scenarios demonstrate that our method achieves state-of-the-art visual consistency in long-duration generation, while enabling complex user-avatar interaction in real time.

## 1. 核心问题与整体含义（研究动机和背景）

**研究动机**：现有基于扩散模型的音频驱动虚拟形象生成方法在实时流式场景中面临两大核心挑战：
- **视觉时间一致性不足**：长时间生成时，由于因果注意力机制局限于少数先前生成块，随着内容逐渐偏离初始参考帧，会出现时间漂移和视觉退化。
- **用户意图感知缺失**：现有方法仅依赖音频驱动口唇同步和简单动作，无法理解用户意图（如“看一下手表并告诉我时间”），无法生成语义对齐的精细动作和场景变化。

**整体含义**：本文旨在构建一个**实时无限流式视频生成框架**，同时实现高度视觉一致性（短程连贯+长程稳定）和意图感知交互（语音+动作+场景动态更新），推动数字人从简单反应式向主动理解式演进。

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想
基于扩散Transformer（DiT），通过**自回归蒸馏**实现实时流式生成，并引入**长-短视觉记忆（LSVM）** 和**推理-反应模块（RRM）**，分别解决一致性和交互问题。

### 2.2 关键技术细节

#### （1）长-短视觉记忆（LSVM）
- **短时记忆**：固定大小的FIFO队列（窗口K），存储最近生成帧的压缩令牌，保证局部连续性。
- **长时记忆**：固定容量N的紧凑令牌集合，存储全局代表性视觉状态。
- **记忆压缩**：轻量级压缩网络C(·)（卷积+注意力），将视频潜变量压缩为令牌，时间压缩比为1（避免跨帧合并）。
- **动态关键帧选择（DKFS）**：当短时记忆弹出帧时，利用SigLIP2提取语义特征，计算长时记忆的全局冗余度R；若替换候选帧后R'<R则保留，否则遗忘。保证长时记忆语义多样性。
- **训练目标**：随机采样目标帧，遮蔽其他帧，以噪声作为掩码，条件为压缩后的长短记忆表示，训练模型重建目标帧。

#### （2）推理-反应模块（RRM）
- **状态循环策略**：
  - 用户语音→ASR→文本→输入LLM（结合当前稳定状态）→输出动作状态（动作指令pact+响应音频aresp）和稳定状态（静态视觉描述pstable）。
  - 生成时，先执行动作状态（音频驱动口唇+动作提示驱动姿态），音频结束后切换为稳定状态（仅用pstable，无音频），直到新指令触发下一个动作循环。
- **缓存切换机制**：当动作提示切换时，重新编码新文本，并重新计算之前相邻块的KV缓存（只刷新受影响的块），减少延迟。

#### （3）自回归蒸馏（DMD）
- 四阶段训练：
  1. 双向注意力训练图像-音频-视频（AI2V）基础模型。
  2. 预训练记忆压缩模块C(·)。
  3. ODE初始化：用块因果注意力近似教师ODE轨迹。
  4. Self-Forcing DMD蒸馏：通过最小化学生与教师诱导噪声分布之间的反向KL散度，将扩散模型蒸馏为少步自回归生成器。

## 3. 实验设计

### 3.1 数据集
- **总规模**：约300万高质量片段（经筛选）。
- **组成**：
  - 语音驱动说话头视频：HDTF、VFHQ、VoxCeleb2、CelebV-Text、AVSpeech。
  - 电影/电视数据：主要来自OpenHumanVid，覆盖复杂场景和长时间动态。
  - 专有对话数据集：长时间说话片段，含丰富身体动作（手势、头肩动态）。

### 3.2 测试场景与基准
- **测试集**：500个视频，包含简单短交互到复杂长视频交互，并在指定时间点触发动作切换。
- **对比方法**：StableAvatar、OmniAvatar、HYAvatar、Hallo3、EchoMimicV3、WanS2V、LiveAvatar（共7个，其中LiveAvatar为实时方法，其余为非实时处理后拼接）。

## 6. 主要结论与发现

- **视觉一致性**：在OBJ（物体一致性）和TV（文本-视频对齐）上达到SOTA（85.2和25.93），ID（身份保持）也表现优异，说明LSVM有效缓解长时间漂移。

- **实时性**：FPS达26.68，高于所有对比方法（LiveAvatar为21.94），DMD蒸馏是关键加速因素（无蒸馏仅1.27 FPS）。

## 7. 优点

1. **首次实现视觉一致且意图感知的实时无限流式交互头像生成**，解决了长期被忽视的一致性漂移问题。
2. **系统架构完整**：LSVM兼顾局部和全局记忆，DKFS基于语义冗余度选择关键帧，避免了简单均匀采样。
3. **RRM设计精巧**：状态循环使模型能在动作-稳定状态间平滑切换，缓存切换机制降低提示变更延迟。
4. **实验充分**：对比方法覆盖主流开源模型，消融实验验证了每个核心设计的必要性。
5. **工程优化**：使用FSDP、KV缓存、流水线并行等技术实现高推理速度，并给出明确的训练超参数。
